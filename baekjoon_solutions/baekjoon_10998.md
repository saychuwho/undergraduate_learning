# baekjoon_1001

## Problem

Get two integer A, B and print the multiplication of A, B. (0 < A, B < 10)

## Idea  

Get two integer A, B with console input and print the multiplication of A, B with console output.

## C++ Solution

```c++
#include <iostream>

int main() {
    int a = 0, b = 0;
    std::cin >> a >> b;
    std::cout << a * b;
}
```
